# Smart Event Dashboard

## Overview

The Smart Event Dashboard is a simple web application that allows users to create and manage events.
This project was built using **HTML, CSS, and JavaScript** without any external libraries.

The application demonstrates **DOM manipulation, event handling, and dynamic content creation** in JavaScript.

## Features

* Add a new event with title, date, category, and description
* Dynamically display events on the dashboard
* Highlight important events
* Delete events
* Clear all events
* Add a sample event for testing

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
